# Portfolio Next.js

Site portfolio moderne construit avec Next.js.

## Installation

1. Ouvrez un terminal dans `c:\Users\ISABELLE\Documents\portfolio`
2. Exécutez `npm install`
3. Lancez le projet avec `npm run dev`
4. Ouvrez `http://localhost:3000`

## Personnalisation

- Remplacez le contenu de `app/page.js` avec vos informations de CV
- Ajoutez vos projets, expériences et compétences
- Changez les couleurs dans `app/globals.css`
